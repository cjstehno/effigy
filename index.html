Effigy Entity & Mapper Framework

Effigy: 1. a sculpture or model of a person. 2. a roughly made model of a particular person, made in order to be damaged or destroyed as a protest or expression of anger

You can decide which definition above fits with your standard data-access experience, but I generally find #2 to be appropriate when trying to do simple quick work.

Features

Effigy provides two primary features:

  • Sql annotations to inject Spring-JDBC code into abstract methods in a simple and consistent manner using minimal code.
  • Entity framework annotations which provide a simple and lightweight entity CRUD framework.
See the User Guide for in-depth documentation about all of the supported features.
