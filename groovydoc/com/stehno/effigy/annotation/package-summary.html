Package com.stehno.effigy.annotation

  • Annotation Type Summary 
    Annotation Type Description
    Association Annotation used to denote that a collection property (Collection implementation) has an association relationship with another entity (based on a reference table).
    Column Annotation used to provide additional information related to the database column represented by the annotated property.
    Component Annotation used to denote a property as a one-to-one relationship with the enclosing entity.
    Count Annotation used to denote a count method in an Effigy repository.
    Create Annotation used to denote a create method in an Effigy repository.
    Delete Annotation used to denote a delete method in an Effigy repository.
    Embedded Annotation used to denote an embedded component field.
    Entity Annotation used to denote an Effigy entity object.
    Exists Annotation used to denote an "exists" method in an Effigy repository.
    Id Denotes an Effigy entity identifier field.
    Limit Annotation used to denote that the annotated parameter is to be used as the limit for the query.
    Mapped Augments a Map property annotated with the @Association annotation to enable additional meta configuration.
    Offset Annotation used to denote that the annotated parameter is to be used as the offset for the query.
    PreparedStatementSetter This annotation is used with a SqlSelect or SqlUpdate annotation to provide information about the PreparedStatementSetter instance to be used.
    Repository Annotation used to denote a repository managed by the Effigy API.
    ResultSetExtractor This annotation is used with a SqlSelect annotation to provide information about the ResultSetExtractor instance to be used.
    Retrieve Annotation used to denote a "retrieve" method in an Effigy repository.
    RowMapper This annotation is used with a SqlSelect annotation to provide information about the RowMapper instance to be used.
    SqlSelect Annotation used to denote a custom SQL-based select query method in an Effigy repository.
    SqlUpdate Annotation used to denote a custom SQL-based update method in an Effigy repository.
    Transient Annotation used to denote that an entity property should be ignored by the mapping framework.
    Update Annotation used to denote an "update" method in an Effigy repository.
    Version Denotes an Effigy entity version field.
