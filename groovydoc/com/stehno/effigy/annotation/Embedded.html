Package: com.stehno.effigy.annotation

[Groovy] Annotation Type Embedded

    • com.stehno.effigy.annotation.Embedded

    • Annotation used to denote an embedded component field. The fields of the embedded object are part of the enclosing table. The type of the embedded object may be an Effigy Entity; however, it need not be. Also, if it is annotated with @Entity, any @Id or @Version annotations will not be honored, since the table data is contained in the table for the enclosing entity.
