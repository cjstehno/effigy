Package com.stehno.effigy.transform

Package com.stehno.effigy.transform Description

The contents of the "com.stehno.effigy.transform" package and its sub-packages are considered "internal use only" and should not be used outside of the Effigy framework itself.
  • Interface Summary 
    Interface Description
    RepositoryMethodVisitor Defines a specialized AST node visitor used by the repository transformer.
  • Class Summary 
    Class Description
    AnnotatedMethod A helper object for bundling information about an annotated method.
    AssociationSaveMethodInjector Generates the entity association saving method for @Create and @Update annotations.
    ClassManipulationUtils AST class manipulation helper utilities.
    ComponentRowMapperTransformer Transformer used for creating a RowMapper instance for the Component associations of an Entity.
    CountTransformer Transformer used to process the @Count annotations.
    CreateTransformer Transformer used to process the @Create annotation.
    DeleteTransformer Transformer used to process the @Delete annotation.
    EntityResultSetExtractorTransformer Transformer used for creating a ResultSetExtractor instance for the entity.
    EntityRowMapperTransformer Transformer used for creating a RowMapper instance for the entity.
    ExistsTransformer Transformer used to process the @Exists annotations.
    MethodImplementingTransformation Abstract parent class for the Effigy repository method implementation annotation transformers.
    RepositoryTransformer Transformer used for processing the EffigyRepository annotation.
    RetrieveTransformer Transformer used to process the @Retrieve annotations.
    SqlHelperAnnotation Immutable object stub used to represent the Sql helper annotation values during the transformation.
    SqlSelectTransformer Transformer used to process @SqlSelect annotated methods.
    SqlUpdateTransformer Transformer used to process @SqlUpdate annotated methods.
    UpdateTransformer Transformer used to process the @Update annotations.

    Exception Description
    EffigyTransformationException Represents an exception caught and rethrown (or generated) during AST processing.
